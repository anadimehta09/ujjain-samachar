# Ujjain Samachar

Ujjain Samachar is a Full Stack News Portal developed using FastAPI, MongoDB, HTML, CSS, and JavaScript. It allows administrators to upload news, images, videos, reels, and manage team members through an admin panel.

## Features

- Admin Login (JWT Authentication)
- Upload News
- Upload Images & Videos
- Breaking News
- Like & Comment System
- Reels Section
- Team Management
- FastAPI REST APIs
- MongoDB Database

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- FastAPI
- Python
- MongoDB
- PyMongo

## How to Run

1. Install requirements

```
pip install -r requirements.txt
```

2. Start MongoDB

3. Run FastAPI

```
uvicorn main:app --reload
```
